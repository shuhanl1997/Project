Brainstorm & sketch

This is a project I created in IAT102 at Simon Fraser University. My teammate and I have worked on magazine design. We are asked to create a magazine cover and two pages of the magazine. For the first step, we decided the topic of the magazine is an architecture mag which the issue name is “Hold the Art”. Then we brainstormed some magazine layouts and created three sketches for each idea. After we finished to sketch out the magazine ideas, we started our first mockup and try the ideas. During this step, we found it was really hard to find images in which the resolution is highenough to make the magazine seems clear enough for our audience so we spent some time on finding high-quality illustrations.



Choose the font

The software is used for this project is InDesign and Lightroom. In the beginning, we were not familiar with either of this software so my teammate and I have searched a lot of tutorials on lynda.com to learn how to use the tools. We also attended all the labs and asked for our professor and TA’s advice. At this stage, we started finding the font we want to use for the magazine. Since the topic we chose is an architecture mag, we used three levels of text and font styles to set the overall style of the magazine. For the headings, we used the transitional font, Sans serif or serif fonts. For the sub-headings, we used the humanest font, Sans serif or serif fonts and the body text using the same fonts as the headings.



First mockup

For the third stage, we officially began to make the logo and the magazine in InDesign. We used software such as PS and AI to cut the images and add effects to the images. As we have learned in the lecture and lab about the margin, white space and some other technical strategies, we were always aware of the details of the magazine page so the mag will look visually better and more consistent.

Final work

In conclusion, the whole working process is very fun and the toughest part was to learn the new design tools. Graphic designers must learn new tools quickly and efficiently and make the product neat to present to the users. I am really glad to learn this course and make this project with my teammate
